StudySync

Project Description:
StudySync is a Smart Student Study Planner developed using HTML, CSS, and JavaScript. It helps students organize tasks, track upcoming exams, and monitor their academic progress through an easy-to-use interface.

Features:
* Add and delete tasks
* Mark tasks as completed
* Search tasks
* Track upcoming exams
* Exam countdown
* Progress tracking
* Responsive design
* Local Storage support

Technologies Used:
* HTML
* CSS
* JavaScript
* Local Storage
